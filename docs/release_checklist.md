# Public release checklist

- [ ] Confirm package name availability on PyPI.
- [ ] Confirm MIT license with all authors and institution.
- [ ] Replace repository URL in `CITATION.cff` and `pyproject.toml` metadata.
- [ ] Run the full Stage-10 regression benchmark.
- [ ] Run unit tests on Python 3.10, 3.11, and 3.12.
- [ ] Inspect wheel contents; exclude proprietary Web of Science data.
- [ ] Create GitHub release and archive it on Zenodo.
- [ ] Add software DOI to the manuscript/data-availability statement.
